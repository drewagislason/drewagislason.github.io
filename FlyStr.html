A set of C string utilities to augment string.h

Let's face it: C is a poor language for string manipulation. This module attempts to make it easier.

Features:

  • Python-like safe strings: FlyPyStrNew(), FlyPyStrCat(), FlyPyStrCheck()
  • Line parsing: FlyStrLineNext(), FlyStrLinePrev(), FlyStrLineLen()
  • Case parsing: lowercase, UPPERCASE, camelCase, MixedCase, snake_case, CONSTANT_CASE
  • Path handling: FlyStrPathAppend(), FlyStrPathParent(), FlyStrPathExt()
  • Dump text/binary: FlyStrDump(), FlyStrDumps()
  • Easy string building: FlyStrZCat(), FlyStrZFill()
  • Copy/compare arrays of strings
  • Many other string utility functions

FlyStrZ...() functions have a few properties:

  1. They always result in an ASCIIZ ('\0' termianted) string
  2. They always fit in the # of bytes specified as size
  3. They can have a NULL szDst, which means just calculate size

One use of FLyStrZ..() is to determine size of strings so you can allocate them. Set the first parameter (szDst) to NULL.

Example: Use FlyStrZCat() to create `["Hello","World","!"]`

#include "FlyStr.h" 

size_t MyAllocArray(char *szDst, size_t size, const char *asz[]) 
{ 
  const char  **ppsz    = asz; 
  size_t        lenDst  = 0; 
  bool_t        fFirstTime = TRUE; 

  lenDst += FlyStrZCpy(szDst, size, "["); 
  while(*ppsz) 
  { 
    lenDst += FlyStrZCat(szDst, size, fFirstTime ? "\"" : ",\""); 
    lenDst += FlyStrZCat(szDst, size, *ppsz); 
    lenDst += FlyStrZCat(szDst, size, "\""); 
    fFirstTime = FALSE; 
    ++ppsz; 
  } 
  lenDst += FlyStrZCat(szDst, size, "]"); 

  return lenDst; 
} 

int main(void) 
{ 
  const char *ArrayOfStrings[] = { "Hello", "World", "!", NULL }; 
  char    *szDst; 
  size_t  len; 

  len = MyAllocArray(NULL, SIZE_MAX, asz); 
  szDst = malloc(len + 1); 
  if(szDst) 
  { 
    len = MyAllocArray(szDst, len + 1, ArrayOfStrings); 
    printf("%s\n", szDst); 
  } 
}

Resulting string will be lowercase alnum, dashes and dots only. Examples:

"  I Love Waffles  "      -> "i-love-waffles"
"my.echo My Shadow & Me"  -> "my.echo-my-shadow-me"

FlyStrNTol

Convert from a string to some base. Similar to strtol(), or atol(), but returns # of digits found.

Back to top

Prototype

long FlyStrNTol(const char *sz, unsigned base, unsigned len, unsigned *pDigits)

@param sz pointer to a string
@param base 2-36
@param len maximum length in sz to look for digits
@param pDigits return value, # of digits found (0-len)
@return TRUE if c is a slash

Notes

The base is limited to 36 because the higher digits use alphabetic characters A-Z. However, for dozenal (base 12), the digits A,B or X,E (Dec, El) can be used.

Case of digits is ignored. The hex (base 16) e2f is same as E2F.
