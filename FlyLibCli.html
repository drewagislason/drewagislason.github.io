- A C API for handling the command-line (argc, argv) in a consistent way.

FlyCli does not FORCE any option convention, but commonly single character options can be combined. For example, "-a" and "-v" are both single character options, and "-av" is both. If a multi-character option is desired, then two dashes are required: "--fancy_arg".

Features:

  • Options can be anywhere in the arglist
  • Will print error messages (or not) based on settings in
  • Get arguments in any order by index
